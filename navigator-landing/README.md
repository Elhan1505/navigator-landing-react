# Лендинг "Система Навигатор"

Современный одностраничный лендинг для Telegram-бота профессиональной ориентации.

## Технологический стек

- **React 18+** с TypeScript
- **Vite** — быстрый инструмент сборки
- **Tailwind CSS 3.4** — утилитарный CSS-фреймворк
- **Framer Motion** — анимации
- **Lucide React** — иконки

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшен-сборки
npm run preview
```

## Структура проекта

```
src/
├── components/ui/      # UI-компоненты (Button)
├── components/layout/  # Layout-компоненты (Section, Container)
├── sections/           # Секции лендинга (Hero, Problem, Process, etc.)
├── data/content.ts     # ЕДИНЫЙ источник всего контента
└── lib/utils.ts        # Утилиты
```

## Редактирование контента

Все тексты находятся в `src/data/content.ts`. Измените текст там — изменения применятся автоматически.

## Секции лендинга

1. **Hero** — первый экран с главным предложением
2. **Problem** — описание проблемы выгорания
3. **Process** — 6 шагов работы системы
4. **Pricing** — тариф 1500₽
5. **FAQ** — 9 частых вопросов
6. **Final CTA** — финальный призыв к действию
